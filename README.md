# payment-reconciliation
２つのCSV（社内データ・取引先データ）を突合し、差異をxlsxで出力する支払いデータ照合ツール。

## なぜ作ったか
手作業で行なっている経理業務の支払データ照合を自動化するため。
「片方にしかないデータ」「金額の不一致」の検出が目的。

## 機能
- 支払Noをキーに2つのCSVを外部結合で突合
- 片方にのみ存在する行を検出
- 金額不一致の行を検出
- 結果をExcel(シート別)で出力

## 使い方
```
pip install pandas openpyxl
python3 shogo.py
```
